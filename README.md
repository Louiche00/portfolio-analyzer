Portfolio Analyzer 📊

Automated portfolio analysis tool for Canadian equities.
Built with pure Python — no external libraries required.

## What it does
- Calculates return (%) for each position
- Computes total portfolio value
- Identifies best and worst performers

## How to run
```bash
python portfolio_analyzer.py
```

## Skills demonstrated
- Python data structures (lists, dicts)
- Functions & modular code design
- Financial return calculations
- Formatted terminal output
